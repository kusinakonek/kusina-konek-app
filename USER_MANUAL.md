# KusinaKonek User Manual

Last updated: 2026-04-03

## 1. What This Guide Covers

This manual is for app users and explains:

1. How to sign up, log in, and reach the homepage.
2. How to use the app as a Donor.
3. How to use the app as a Recipient.
4. How to manage profile, notifications, chat, and common issues.

## 2. Quick Flow (Login to Homepage)

First-time user flow:

Welcome -> Sign Up -> Verify Code (OTP) -> Login -> Homepage

Returning user flow:

Open App -> Auto Session Check -> Homepage

If not logged in, the app opens the Welcome/Login screens.

## 3. Before You Start

For best experience, allow these permissions when prompted:

1. Notifications: for claim and message alerts.
2. Location: for nearby food and map features.
3. Camera: required when posting a food donation.

## 4. Account and Authentication

## 4.1 Create an Account (Sign Up)

1. Open the app.
2. Tap Sign Up.
3. Enter your details:
   - Email
   - Password
   - Full name
   - Barangay
   - Phone number
   - Role (Donor or Recipient)
4. Submit sign up.
5. Check email for OTP code.
6. Enter OTP in Verify screen.
7. After verification, your profile is created and app opens main area.

## 4.2 Log In

1. Tap Login.
2. Enter email and password.
3. Choose role for this session:
   - Donor
   - Recipient
4. Tap Login.
5. You are redirected to homepage.

## 4.3 Forgot Password

1. Tap Forgot Password.
2. Enter your email.
3. Enter verification code from email.
4. Set a new password.
5. Return to Login and sign in.

## 4.4 Logout

1. Open Profile.
2. Tap Logout.
3. App returns to Welcome screen.

## 5. Homepage and Navigation

After login, homepage depends on your selected role:

1. Donor: Donor Dashboard
2. Recipient: Recipient Dashboard

Main navigation tabs:

1. Home
2. Cart (Recipient only)
3. Profile

Additional pages are opened from buttons inside screens:

1. Notifications
2. Browse Food
3. Donate flow
4. Chat
5. Edit Profile

## 6. Donor User Manual

## 6.1 Post a Food Donation

From Donor Homepage:

1. Tap Donate Food.
2. Step 1: Select food type (preset or custom).
3. Step 2: Enter food details:
   - Food name
   - Description
   - Quantity (servings)
   - Availability duration
   - Required photo of actual food
4. Step 3: Set drop-off location:
   - Choose Barangay Hall, or
   - Pin custom location on map
5. Tap Confirm and Submit Donation.
6. Success message appears, then return to dashboard.

## 6.2 What Happens After Posting

1. Donation becomes available to recipients.
2. Nearby recipients receive notification.
3. Your donation appears in recent/active list.

## 6.3 Manage Active Donation

From Donor Home or Active Details:

1. View donation status.
2. Open chat once a recipient claims.
3. Cancel donation if still available and eligible.

## 6.4 View Feedback

1. Open Feedback screen.
2. View ratings and comments from recipients.
3. Open review details for each completed donation.

## 7. Recipient User Manual

## 7.1 Find Available Food

From Recipient Homepage:

1. Tap Browse Food to view list.
2. Use Food Map to view location pins.
3. Open food details for servings, location, and donor info.

## 7.2 Add to Cart and Claim

1. Add food item to cart.
2. Open Cart tab.
3. Review selected items.
4. Confirm pickup request.
5. Successful claims move to your recent claimed list.

Important:

1. Cart reservations are time-limited (about 15 minutes locally).
2. If item expires or is no longer available, it is removed from claim flow.

## 7.3 Pickup Status Actions

For claimed food, status usually moves as:

PENDING -> CLAIMED -> ON_THE_WAY -> COMPLETED

Recipient actions:

1. Mark On The Way when traveling to pickup.
2. Confirm Receive after getting the food.
3. Submit feedback after completion.

## 7.4 Submit Feedback

1. Open completed donation.
2. Tap feedback option.
3. Add rating and comment.
4. Optionally attach photo.
5. Submit.

## 8. Chat and Messaging

1. Chat opens per donation/distribution.
2. Donor and recipient can exchange text and images.
3. Message unread counts appear on relevant cards.
4. Messages are marked read when chat is opened.

## 9. Notifications

You may receive notifications for:

1. New food near you.
2. Donation claimed.
3. Recipient on the way.
4. Delivery confirmation.
5. New message.
6. Feedback reminders and system updates.

To view all alerts:

1. Open Notifications page from bell icon.
2. Tap an item to follow related action.

## 10. Profile and Settings

From Profile screen:

1. Edit personal information.
2. Switch role (Donor or Recipient).
3. Manage notification preference.
4. Change password (if needed).
5. Delete account (if needed).

## 11. Common Problems and Fixes

Cannot log in:

1. Check email and password.
2. Confirm account is verified.
3. Use Forgot Password if needed.

No notifications:

1. Enable notifications in phone settings.
2. Ensure internet is active.
3. Reopen app and check notification preference.

Cannot claim food:

1. Item may already be claimed.
2. Item may be expired.
3. Claim limits/restrictions may apply.

Chat not updating instantly:

1. Pull to refresh.
2. Check internet connection.
3. Reopen chat screen.

## 12. Best Practices for Users

1. Keep profile details up to date.
2. Keep notifications enabled for faster coordination.
3. For donors, post clear photos and accurate location pins.
4. For recipients, mark statuses promptly and provide feedback.
5. Use chat for pickup coordination and clarity.

## 13. Support Notes

If a feature behaves unexpectedly:

1. Restart app.
2. Check network quality.
3. Try action again after refresh.
4. Report issue with screen name and timestamp for faster support.
